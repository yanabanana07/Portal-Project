About This Project

This is a front-end-only Student Portal demonstration built for educational purposes.

All data is simulated and temporarily stored in your browser's LocalStorage. No information is sent to a server, and no real database is used. Refreshing or clearing your cache may reset the data to its default state.

Technologies Used:
  • HTML5
  • CSS3 (Flexbox, Grid, Variables)
  • JavaScript (ES6+, Modular)
  • Chart.js (for data visualization)
  • jsPDF & html2canvas (for PDF generation)

This project showcases a modern, responsive UI/UX, complex workflows (like exam permit generation), and dynamic data handling entirely on the client-side.
